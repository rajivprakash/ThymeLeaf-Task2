what the different between the new topology and previous one (naming , or terms will be accepted as an answer)?


The previous one is developed based on Include-style layouts and the current task is developed based on Hierarchical-style layouts.

Include-style layouts : 

a) Thymeleaf's th:include allows to pass the entire element fragments to be included in a HTML page. Useful in reusing the part of HTML code.
b) Thymeleaf's th:replace to replace the code by the specified part of HTML code.

Hierarchical-style layouts :

The templates are usually created with a parent-child relation, your parent template(Master.html) that will get filled-up or overwritten by your content pages (Home.html and Contact.html).
